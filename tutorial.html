Tutorial DFA GAME

เมื่อเข้าเกม จะมีปุ่มให้เลือกโหมดในการเล่นมีทั้งหมด 3 โหมด easy nomal hard

ทางซ้ายมือ จะเป็นโจทย์ DFA ให้เลือก ส่วนด้านซ้ายจะเป็นส่วนที่เก็บข้อมูล input ที่ input เข้าไปและตรวจสอบว่า input นั้น Accept หรือ Reject

เมื่อ input string เข้าไปสามารถกดปุ่ม check ได้เพื่อที่จะ เช็คว่า input นั้น appcted หรือ rejected หาก input นั้น accepted จะทำการบันทึก หาก rejected ระบบจะแจ้งเตือนเฉยๆ แต่ไม่ได้ทำการบันทึก input นั้นๆ

หลังจากนั้น ให้เลือกรูป DFA ที่คาดว่าน่าจะถูกต้อง เพื่อทำการตรวจคำตอบ

หากคำตอบนั้นถูกต้องก็จะสามารถไปข้อต่อไปได้ หากผิดสามารถทำใหม่ได้อีกรอบ
